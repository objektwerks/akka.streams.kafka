Akka Streams Kafka
------------------
>Akka Streams and Alpakka Kafka Connector test app using Scala 3.

Note
----
>In this project, Akka and Akka Streams works with Scala 3. But in
>my Akka Streams project (https://github.com/objektwerks/akka.streams)
>it does not.

Run
---
1. sbt clean run
